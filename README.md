# 迷你游戏合集

这个文件夹里有两个自包含的浏览器小游戏，直接打开 HTML 文件就可以玩。

## 游戏

- `snake-game.html`：霓虹贪吃蛇。
- `ball-arena.html`：星团大作战，类球球竞技玩法。

## 霓虹贪吃蛇亮点

- 三个难度：轻松、街机、高手。
- 关卡推进：每吃 5 个食物进入下一关。
- 街机和高手模式会生成岩块障碍。
- 金果、蓝晶、护盾三类特殊道具。
- 连击、狂热、慢动作、粒子反馈和本地最高分。
- 支持键盘、WASD、触屏滑动和屏幕方向键。

## 星团大作战亮点

- 开始界面可选择自由模式、团队战、生存模式和大逃杀。
- 自由模式：100 人同场，12 分钟限时，死亡后复活，按个人质量结算。
- 团队战：10 队 5 人，12 分钟限时，队友可接分身，按队伍总质量结算。
- 生存模式：9 分钟生命制，吞噬可增加生命，按击杀、剩余生命和质量排名。
- 大逃杀：100 人开局，一命制，安全区持续收缩，活到最后获胜。
- 7600 像素大地图，后期食点刷新更快，战场会越来越密。
- 最大 16 个分身，支持分裂突袭、长按 W 连续吐球和冷却后自动合体。
- 有小刺和大刺两类刺球，触碰后会消失并重生，炸刺会先增加质量再分裂。
- AI 会追击、逃跑、进圈、分裂攻击、吐球、借刺反打、围猎、护航、接队友分身并给主力喂球。
- 后期会按可见区域绘制食点和细胞，并自动降低部分阴影、标签、小地图刷新和粒子细节来稳定帧率。
- 全屏竞技场、排行榜、小地图、粒子反馈和移动端触控。

## 霓虹贪吃蛇操作

- 方向键 / WASD：移动。
- 空格：暂停或继续。
- 点击右上角按钮：暂停、静音、重新开始。

## 星团大作战操作

- 鼠标或手指：控制方向。
- 空格：分裂。
- W：按住连续吐球。
- P：暂停或继续。
- R：重新开始。

## 运行

双击对应的 HTML，或在这个目录启动一个简单本地服务：

```bash
python -m http.server 8000
```

然后打开：

- `http://localhost:8000/snake-game.html`
- `http://localhost:8000/ball-arena.html`
