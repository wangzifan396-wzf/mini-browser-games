# 迷你游戏合集

100 款零依赖、单 HTML 文件的浏览器小游戏。每个页面都可以直接双击运行，游戏逻辑、界面与绘制代码均封装在对应 HTML 中；重点局外成长作品支持导出、导入跨设备存档码。

## 最新更新

- 《弹弓风暴》升级为风眼 V4：三章 18 个物理节点、三场风眼首领、独立挑战三星、五系风暴工坊、里程碑、程序化音效与跨设备档案组成完整远征。
- 《霓虹贪吃蛇》升级为星环 V4：三章 18 个独立节点、双挑战三星、章节连续构筑、三场多层核心首领、断点续航与程序化音效共同组成完整远征。
- 《霓虹 2048》升级为星图 V4：六章 18 个独立节点、双挑战三星、协议构筑、八件救盘工具、危急救盘与自动续局把经典合成扩展为长线玩法。
- 《四季小农场》加入四季 16 种作物、居民关系、节庆、订单、建筑、三种农场出身和完整年度循环。
- 《前线指挥部》升级为三战线实时战略，包含六兵种、六类建筑、三种指挥学说、五章战役和永久研究。
- 《雾城侦探》扩展为六起连续案件，需要搜查现场、盘问证人、关联证据并完成凶手/动机/手法三重指控。
- 《午夜监控室》加入六个故事夜、五类不同行为的异常体、九路监控、设备升级和电力/信号双资源管理。
- 《口袋地铁》加入五座规则不同的城市、动态车站、换乘寻路、桥隧限制、车辆分配和每周升级。
- 上述作品均支持桌面与触屏操作、自动保存及带校验的跨设备存档码。全部评级与后续更新顺序见 [100 款游戏质量审计](GAME_AUDIT.md)。

## 如何运行

直接双击任意 `.html` 文件即可。若浏览器在本地文件模式下限制存档，建议在项目目录启动本地服务：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000/文件名.html`，例如 `http://localhost:8000/ball-arena.html`。
