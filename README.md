这个文件夹里有五十个自包含的浏览器小游戏，直接打开 HTML 文件就可以玩。

## 新增热门街机游戏亮点

- 雷霆先锋：自动火力竖版射击，包含能量核心、波次首领和三选一武器协议；支持键盘与拖动操控。
- 方块熔炉：标准十列落块、七袋随机、幽灵落点、暂存、硬降与连消等级加速。
- 深空扫雷：新手/标准/硬核三种棋盘，首击安全、标旗、数字连开、限量扫描提示和本地最佳时间。
- 棱镜破阵：拖动挡板反射光球，棱镜砖会提供多球、扩展或激光能力，砖阵随关卡加密。
- 云端跃迁：自动弹跳的纵向平台玩法，收集星片、踩弹簧与滑翔云，平台会随着高度逐渐变窄。
- 爆格特攻：格子爆破对抗，包含砖墙、范围炸弹、巡逻敌人、生命、关卡和三类强化道具。
- 果切风暴：60 秒动作切割，连续一刀切多个水果提高连击，炸弹与漏果都会损失机会。
- 数独工坊：可点击或键盘填入数字，错误即时高亮，支持擦除、限量提示与多张标准题面。
- 四子战线：七列落子、四连胜判定、封堵/优先获胜的 AI，以及标准/高手两种决策随机度。
- 点球热浪：六个射门区域、门将随机站位、三档蓄力、连续进球和十轮赛季比分。
- 霓虹冰球：拖动球拍截击冰球，AI 会随比分提升追球速度，五分制赛局适合短局对抗。
- 像素飞翼：单键扇翼穿越风门，每十分快速切换区域，完美过门与金门形成倍率追分。
- 霓虹叠塔：移动方块一键落下，保留重叠区域；连续完美落点可恢复塔宽并挑战本地最佳。
- 五字密语：六次猜词机会，绿色表示位置正确、金色表示字母存在，支持实体键盘和屏幕键盘。
- 麻将连台：选择两个相同且至少一侧开放的麻将牌，支持洗牌、提示与完整清台目标。
- 霓虹迷宫：吃完光豆完成迷宫，大能量豆会让追猎者进入可捕获状态。
- 星球大合成：真实重力和圆形碰撞，相同星球接触后升级，堆过红线即结束。
- 霓虹二十一点：完整筹码循环，支持下注、要牌、停牌和加倍，庄家按规则自动补牌。

## 芽阵前线亮点

- “四境远征”包含 12 个冒险关卡，依次穿越晨露庭院、萤雾湿地、霜风高地与星辉遗迹；通关会解锁芽阵、记录星级并自动保存进度。
- 18 种原创芽阵覆盖日光生产、直射、双发、减速、阻挡、爆破、地刺、抛射、溅射、持续伤害、近战、控制、拆甲、贯穿、连锁与辅助等构筑方向。
- 14 类推进敌人包含疾行、跳跃、盾甲、光环、治疗、掘地、飞行、冲撞、重型、群体与首领单位，并会随区域和波次组成不同队列。
- 每局可从已解锁芽阵中自选最多 8 张卡；关卡外提供芽阵图鉴、推荐卡组、本地进度、三星评价和累计击退记录。
- 叶能最多储存三格，点“叶”后选择芽阵可触发各自的专属绽放；星辉遗迹的辉能格还会联动其他格上的芽阵。
- 除冒险外，提供持续增强的无尽花圃、无需日光的传送育苗以及按日期固定阵容和敌群的今日异象。
- 四个区域分别带来天空日光、浓雾与低日光、周期霜冻、辉能地砖等规则变化；保留卡片冷却、铲除、五路最后防线和波次旗帜等守线节奏。
- 全部代码、美术和进度数据仍封装在单个 HTML 中，支持桌面和触屏布局，可暂停、2 倍速或随时回到远征地图。

## 晶糖花园亮点

- 经典交换三消玩法，点击或拖拽相邻晶糖，三连消除并触发坠落补位。
- 四连会生成横向或纵向爆，五连会生成星爆，连锁越多分数越高。
- 包含步数限制、三色目标收集、提示、洗牌和小锤道具。
- 鼠标和触屏同一套操作，本地记录最佳分数，适合休闲短局。

## 轨道弹球亮点

- 伪三维透视弹球台，2D 物理模拟球、边墙、撞柱、轨道和挡板反弹。
- 空格或触屏长按蓄力发射，左右挡板可连续救球并打出连撞倍率。
- 掉球、剩余球、发射蓄力、连撞倍率和本地最佳分数形成完整街机反馈。
- 全部绘制和逻辑都在单个 HTML 内，不需要外部物理库或构建步骤。

## 巷馆对决亮点

- 2D 街机格斗玩法，包含移动、跳跃、轻击、重击、格挡、闪避和 AI 对手。
- 攻击有前摇、判定、破防、击退和连击评分，短局对战节奏更明确。
- 体力、气力、倒计时、评分和最佳记录实时显示，胜负后可立即重开。
- 支持键盘和移动端按键，适合桌面试玩和手机触屏操作。

## 粉笔台球亮点

- 拖拽白球反方向蓄力，松手出杆，球体碰撞、边库反弹和袋口判定实时模拟。
- 10 个目标球、白球落袋罚分、连续进球加分和清台奖励构成完整练杆流程。
- 所有球停稳后才允许下一杆，避免误触造成连续击球。
- 支持轻推 / 强力模式、白球复位、本地最佳分数和触屏拖拽。

## 信标车队亮点

- 护送动作策略玩法，车队自动推进，玩家控制护卫清敌、拾取零件和部署信标塔。
- 敌人会从两侧袭来并优先攻击车队，护卫和信标塔会自动瞄准射击。
- 零件可以修复车队、加分并作为额外部署资源，形成取舍节奏。
- 路段、路程、耐久、冷却和最高分实时反馈，短局目标清晰。

## 节拍饭团亮点

- 节奏烹饪玩法，根据节拍把米、苔、鱼、汤按正确顺序送进饭团线。
- 客人订单、灶台热度和耐心共同形成压力，连击越稳分数越高。
- 支持键盘快捷键和屏幕按钮，适合桌面与移动端轻量游玩。
- 本地记录最佳分数，重开后节奏循环不会叠加。

## 影邮潜行亮点

- 回合制潜行解谜，观察巡逻灯光，在有限步数内送完邮件并抵达出口。
- 包含撤回、等待、烟雾弹和多关卡目标，适合短局推理。
- 敌人视野、邮件、出口和障碍都直接画在棋盘中，移动端也有方向按钮。
- 撤回逻辑可从被发现状态恢复，方便尝试不同路线。

## 夜市面摊亮点

- 时间管理玩法，接单、煮面、加浇头、出餐形成多任务节奏。
- 三个锅位、不同顾客耐心和订单要求，考验排队与优先级判断。
- 支持鼠标、触屏和按钮操作，开摊、暂停、重开流程完整。
- 本地最高收入记录，重开不会叠加旧动画循环。

## 棱镜果园亮点

- 光线反射解谜，放置斜镜、反斜镜和分光器点亮果树。
- 多关卡、有限镜片库存和操作计数，鼓励用更少步骤通关。
- 点击已有镜片可替换或翻转，拆除会返还库存，避免误操作卡死。
- 支持点亮检测、重开和移动端触控。

## 字灵守门亮点

- 打字防守玩法，输入逼近城门的字灵来击退敌人。
- 连击、充能击退、门耐久和本地最佳分数构成完整单局反馈。
- 出怪位置经过调整，开局就能进入节奏。
- Canvas 圆角绘制做了半径保护，避免极端尺寸下绘制异常。

## 道岔快线亮点

- 铁路调度玩法，切换 3 个道岔，把不同颜色列车送往对应站台。
- 列车连续发车、错线会造成事故，适合快速判断和预判路线。
- 支持点击道岔、数字键和移动端按钮。

- 三条进攻路线、怪物契约、怪潮队列和战场邪术共同构成编队策略。
- 英雄 AI 会根据上一波怪潮补塔、升级和调整目标，后续夜晚逐步反制。
- 每夜结束可选择异变强化，形成持续进化的怪潮构筑。

## 萝卜边境亮点

- 传统塔防玩法，放置弩塔、泡泡塔、萝卜炮等守住萝卜生命。
- 多地图、多难度、障碍清理、炮塔升级/出售和技能支援。
- 没有敌人时炮塔会自动清理障碍，形成经济和建造节奏。
- 8 波流程、评分结算和规则弹窗，适合轻量单局游玩。

## 霓虹贪吃蛇操作

- 在远征站选择行动模式、蛇型框架和已解锁区段，也可消耗光核升级实验室。
- 方向键 / WASD、触屏滑动或屏幕方向键控制移动；不能直接反向。
- 能量达到框架所需值后按 Shift 或点击“脉冲”，释放清障、相位、狂热或护盾能力。
- 空格暂停或继续，右上角可静音、重开或返回远征站。

## 星团大作战操作

- 鼠标或手指：控制方向，移动端可用左下摇杆。
- 空格：分裂。
- W：按住连续吐球。
- A：霸屏模式快速合球。
- D：霸屏模式破阵冲刺并种下战术刺球。
- P：暂停或继续。
- R：重新开始。
- 厅：返回大厅切换模式或外观。
- 音 / 乐：切换 BGM。
- 商店页可导出 / 导入存档码，用于迁移本地进度。

## 霓虹装甲操作

- WASD / 方向键：移动。
- 移动端左下摇杆：移动。
- 鼠标或手指：瞄准。
- 鼠标左键 / 空格：开火。
- 鼠标右键 / Shift / E：释放电磁脉冲。
- 开局可组合底盘和主炮；底盘会把技能替换为电磁脉冲、截击超频、堡垒壁垒或工程支援。
- B：在竞技模式中提示模块自动装配规则。
- P：暂停或继续。
- R：重新开始。

## 霓虹 2048 操作

- 方向键 / WASD / 滑动：移动方块。
- 锤子、炸弹、交换和升阶需要选择工具后点击目标格；角落磁吸和撤回会立即生效。
- 协议充能达到 100% 后选择一项局内强化，也可转换成随机工具。
- Esc 取消当前工具，Z 快速撤回一步；顶部按钮可重开或返回矩阵站。

## 山海伏魔录操作

- 点击地图节点推进流程。
- 点击手牌出牌，点击结束回合。
- 战斗后点击奖励、事件或歇脚选项继续。

## 薯仔突围操作

- 在行动营地选择战薯、区域和危险度，也可消耗结算晶薯购买永久强化。
- WASD / 方向键移动；按住鼠标或触控战场可朝对应方向移动，移动端也可使用左下摇杆。
- Shift 或移动端“冲刺”按钮可短距离突进并获得短暂无敌；Q 或右下“战技”按钮在能量满后释放角色专属能力。
- 武器自动索敌；升级时从三项中选择，也可重抽或转化为治疗。靠近经验、晶薯、补给和遗物箱会自动拾取。
- 空格或 P 暂停，顶部可切换 1 倍/2 倍速度、重开当前配置或返回行动营地。

## 怪潮逆塔防操作

- 选择出怪路线，再点击怪物契约加入队列。
- 智能配队可自动填充队列，释放怪潮后观察英雄防线反应。
- 战场邪术可在选择路线后释放，夜晚结束后选择异变强化。
- 点击规则查看玩法说明，新局可重置流程。

## 萝卜边境操作

- 点击空地建塔，点击已有炮塔升级或出售。
- 开波前可切换地图和难度，开波后守住萝卜生命。
- 技能按钮可选择位置释放，速度和暂停按钮控制战斗节奏。
- 新局和规则按钮位于顶部。

## 芽阵前线操作

- 在远征地图选择关卡，再从已解锁芽阵中选择 3–8 张出战卡；特别行动页可进入无尽、传送带或每日模式。
- 点击卡片后点击草坪格部署；点击黄色日光补充资源，绿色叶滴补充叶能。护壳瓜需要覆盖在已有芽阵上。
- 叶能每满 100 可释放一次：点击“叶”后再点已有芽阵，触发该芽阵的专属绽放。
- 点击“铲”后再点格子会移除格内芽阵；“1×/2×”切换战斗速度，问号可查看当前区域机制。
- 数字键 1–8 快选卡片，空格或 P 暂停，Esc 取消铲除或叶能模式；鼠标与触屏都可直接操作。

## 弹弓风暴操作

- 鼠标或触屏拖拽弹丸蓄力，松开发射。
- 飞行中点击技能按钮或按空格触发当前弹丸技能。
- 点击换风按钮或按 Q 切换风向，P 暂停，R 重开。

## 晶塔前线操作

- WASD / 方向键移动，点击战场设置移动点或锁定敌方目标。
- Q 弧光刃、W 相位步、E 星沼、R 晶陨，侧栏技能按钮也可释放。
- 点击补给按钮购买强化，空格或 P 暂停，顶部按钮可重开。

## 漂移赛道操作

- W / 方向上：加速。
- A / D 或方向左 / 方向右：转向。
- S / 方向下：制动。
- Shift / 空格：氮气。
- P：暂停或继续，顶部按钮可暂停和重开。

## 松果快跑操作

- A / D 或方向左 / 方向右：换线。
- W / 方向上 / 空格：跳跃。
- S / 方向下：下滑。
- P：暂停或继续，触屏按钮可换线、跳跃和下滑。

## 雪岭快逃操作

- A / D 或方向左 / 方向右：空中旋转。
- W / 方向上 / 空格：跳跃或旋转。
- S / 方向下：压低重心。
- Shift：冲刺。
- P：暂停或继续，触屏按钮可旋转、跳跃和冲刺。

## 节拍饭团操作

- A / S / D / F：选择米、苔、鱼、汤。
- 点击底部食材按钮也可操作，节拍命中越准连击越高。
- P 暂停，R 重开，顶部按钮也可控制流程。

## 影邮潜行操作

- WASD / 方向键或屏幕方向按钮：移动邮差。
- 等待：原地过一回合；烟雾弹：本回合无视灯光。
- 撤回可退回上一步，重开可重置当前关卡。

## 夜市面摊操作

- 点击订单、锅位、浇头和出餐按钮完成订单。
- 数字键可快速选择锅位，顶部按钮可开摊、暂停或重开。
- 移动端直接点按各控件即可游玩。

## 棱镜果园操作

- 选择镜片工具后点击棋盘格放置或替换镜片。
- 拆除工具会返还镜片库存，点亮按钮用于检查光路。
- R 重开当前关卡。

## 字灵守门操作

- 直接键入画面中出现的字词来击退字灵。
- 点击击退按钮可释放充能击退，R 重开。
- 优先输入最靠近城门的目标。

## 道岔快线操作

- 点击画面道岔或按 1/2/3 切换方向。
- 红、蓝、绿列车要进入对应颜色站台。
- P 暂停，R 重开。

## 潮汐拾荒操作

- WASD / 方向键或屏幕方向按钮：移动潜水员。
- Shift 或冲刺按钮：短时冲刺。
- 回到水面会自动卖出货物并补满氧气。

## 像素谜格操作

- 选择填黑、画叉或擦除工具后点击格子。
- 鼠标右键可快速画叉，检查按钮可手动判定。
- 完成后可以进入下一题。

## 彗尾织者操作

- WASD / 方向键移动，Shift 或闪现按钮闪现。
- 移动端可拖动画布控制位置。
- 贴近弹幕擦弹得分，击破 Boss 获胜。

## 月面轻降操作

- ↑ / W：主推；←/→ 或 A/D：转向；↓ / S：自动扶正。
- 移动端可用底部按钮控制。
- 低速、低角度接触平台即可成功着陆。

## 菜摊敲敲乐操作

- 点击坏菜和小偷得分，别点带笑脸的好菜。
- 连击达到要求后可点专注按钮放慢目标速度。
- 失误满 5 次或时间结束后结算。

## 灯阵记忆操作

- 观察灯阵闪烁后，按同样顺序点回去。
- 键盘 1/2/3/4 对应四盏灯。
- 重放会扣少量分数，R 可重开。

## 轨道弹球操作

- A / ← 控制左挡板，D / → 控制右挡板。
- 空格或发射按钮按住蓄力，松开发射。
- P 暂停，R 重开，掉球后会自动摆回发射轨道。

## 巷馆对决操作

- A / D 移动，W 跳跃。
- J 轻击，K 重击，L 格挡，Shift 闪避。
- 移动端可用侧栏按钮操作，P 暂停，R 重开。

## 粉笔台球操作

- 在白球附近按下并向后拖拽，松开出杆。
- 所有球停稳后才能下一杆，空格或按钮可摆回白球。
- 轻推 / 强力按钮会调整出杆力度区间，R 重开。

## 信标车队操作

- WASD / 方向键移动护卫，靠近敌人会自动射击。
- 空格或信标按钮部署临时信标塔。
- 拾取零件可修车和加分，P 暂停，R 重开。

## 晶糖花园操作

- 点击两颗相邻晶糖交换，也可按住拖向相邻格。
- 四连和五连会留下特殊晶糖，后续消除会触发爆破。
- 提示会标出可走一步，洗牌扣 1 步，小锤可敲掉 1 颗晶糖。

## 星炉工坊操作

- 点击校准星炉或画布可手动获得星尘，积攒星能后可启动星炉超频。
- 在建筑、研究、委托、专员、远航、战斗、成就、星核分页中购买装置、完成升级和交付资源。
- 远航会在离线时继续计时，返航后带回研究、合金、星能和遗物碎片。
- 星域战斗会自动推进战区，获得战利组件、资源包和碎片，装备越强推进越稳。
- 战斗页可切换突击、稳守和打捞姿态；每 5 个战区会遇到首领，击破后掉落更多战利。
- 达成成就后进入成就页点亮局外星图，星图能力会跨重启保留。
- 研究页可升级研究矩阵；专员页可做专员进阶；战斗页可升级装备和战斗核心，三条养成线会互相消耗并反哺资源。
- 星环页可消耗星辉、数据、碎片、战利和时砂扩建跨模块工程，用来打通后期资源循环。
- 天象页可消耗声望、暗质、星辉和数据校准长期天象，也可以用星域演算把积压资源转换为声望、暗质、时砂和高阶材料。
- 总览页可切换工业、远航、战斗、档案四种星炉协议，资源不足时偏工业，做遗物偏远航，推战区偏战斗，长期离线偏档案。
- 离线回来会获得时砂；点击时间跃迁可消耗 1 时砂快进 10 分钟生产、战斗和远航进度。
- 存档页可输入兑换码，`STARFORGE`、`IDLE2026`、`BATTLELOOP`、`STARMAP`、`WELCOME`、`TIMESHIFT`、`MATRIX`、`STARRING`、`CELESTIAL` 都可领取一次。
- 游戏会自动本地保存，导出 / 导入存档码可把进度迁移到其他设备。
- 星核重启会清空本轮建筑和研究，但保留星核、专员、遗物、成就和星核天赋。
