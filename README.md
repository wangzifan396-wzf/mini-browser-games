# 迷你游戏合集

这个文件夹里有二十五个自包含的浏览器小游戏，直接打开 HTML 文件就可以玩。

## 游戏

- `snake-game.html`：霓虹贪吃蛇。
- `ball-arena.html`：星团大作战，类球球竞技玩法。
- `tank-arena.html`：霓虹装甲，俯视角坦克竞技。
- `neon-2048.html`：霓虹 2048，道具充能版数字合成。
- `shan-hai.html`：山海伏魔录，卡牌肉鸽原型。
- `spud-arena.html`：薯仔突围，自动射击肉鸽。
- `monster-horde.html`：怪潮逆塔防，反向塔防编队。
- `radish-guard.html`：萝卜边境，传统守塔防。
- `drift-racer.html`：漂移赛道，俯视角 2D 漂移竞速。
- `forest-dash.html`：松果快跑，林间三线跑酷。
- `snow-ridge.html`：雪岭快逃，雪坡滑行竞速。
- `slingstorm.html`：弹弓风暴，原创弹弓物理解谜。
- `moba-frontier.html`：晶塔前线，轻量级原创 MOBA。
- `beat-bento.html`：节拍饭团，节奏烹饪反应游戏。
- `shadow-post.html`：影邮潜行，回合制潜行送信解谜。
- `noodle-shift.html`：夜市面摊，订单节奏时间管理。
- `prism-orchard.html`：棱镜果园，镜片反射光线解谜。
- `glyph-warden.html`：字灵守门，中文打字防守。
- `switchyard-rush.html`：道岔快线，铁路道岔调度。
- `tide-salvage.html`：潮汐拾荒，氧气管理潜水探索。
- `pixel-clues.html`：像素谜格，轻量数织逻辑谜题。
- `comet-weaver.html`：彗尾织者，弹幕闪避 Boss 战。
- `moon-lander.html`：月面轻降，重力物理着陆。
- `mole-market.html`：菜摊敲敲乐，反应点击辨认目标。
- `lantern-memory.html`：灯阵记忆，灯序记忆挑战。

## 节拍饭团亮点

- 节奏烹饪玩法，根据节拍把米、苔、鱼、汤按正确顺序送进饭团线。
- 客人订单、灶台热度和耐心共同形成压力，连击越稳分数越高。
- 支持键盘快捷键和屏幕按钮，适合桌面与移动端轻量游玩。
- 本地记录最佳分数，重开后节奏循环不会叠加。

## 影邮潜行亮点

- 回合制潜行解谜，观察巡逻灯光，在有限步数内送完邮件并抵达出口。
- 包含撤回、等待、烟雾弹和多关卡目标，适合短局推理。
- 敌人视野、邮件、出口和障碍都直接画在棋盘中，移动端也有方向按钮。
- 撤回逻辑可从被发现状态恢复，方便尝试不同路线。

## 夜市面摊亮点

- 时间管理玩法，接单、煮面、加浇头、出餐形成多任务节奏。
- 三个锅位、不同顾客耐心和订单要求，考验排队与优先级判断。
- 支持鼠标、触屏和按钮操作，开摊、暂停、重开流程完整。
- 本地最高收入记录，重开不会叠加旧动画循环。

## 棱镜果园亮点

- 光线反射解谜，放置斜镜、反斜镜和分光器点亮果树。
- 多关卡、有限镜片库存和操作计数，鼓励用更少步骤通关。
- 点击已有镜片可替换或翻转，拆除会返还库存，避免误操作卡死。
- 支持点亮检测、重开和移动端触控。

## 字灵守门亮点

- 打字防守玩法，输入逼近城门的字灵来击退敌人。
- 连击、充能击退、门耐久和本地最佳分数构成完整单局反馈。
- 出怪位置经过调整，开局就能进入节奏。
- Canvas 圆角绘制做了半径保护，避免极端尺寸下绘制异常。

## 道岔快线亮点

- 铁路调度玩法，切换 3 个道岔，把不同颜色列车送往对应站台。
- 列车连续发车、错线会造成事故，适合快速判断和预判路线。
- 支持点击道岔、数字键和移动端按钮。
- 分数、到站数、事故数和最佳记录实时显示。

## 潮汐拾荒亮点

- 潜水探索玩法，管理氧气和船舱容量，下潜拾荒后回水面卖出。
- 冲刺会消耗氧气，深度、估值和货物数量共同影响风险收益。
- 支持方向键、WASD、Shift 冲刺与触屏方向按钮。
- 水面自动结算并补氧，适合一局一局刷更高估值。

## 像素谜格亮点

- 数织逻辑谜题，依据行列提示填黑、画叉或擦除格子。
- 多个小关卡，支持检查、下一题和自动完成判定。
- 右键可快速画叉，移动端可用工具按钮切换操作模式。
- 操作数、错误数和最佳记录帮助追求更优解。

## 彗尾织者亮点

- 弹幕闪避 Boss 战，移动、擦弹和闪现构成核心节奏。
- Boss 血量、护盾、擦弹分数和闪现冷却形成紧张反馈。
- 支持键盘、方向按钮和触屏拖动画布移动。
- 弹幕密度随时间推进，短局挑战感更强。

## 月面轻降亮点

- 重力物理着陆玩法，控制推力和角度，把登月舱稳稳降到平台。
- 速度、角度、燃料和评分实时反馈，失败与成功条件清晰。
- 支持方向键、WASD、自动扶正和移动端按钮。
- 地形与平台绘制在同一画布中，适合直接打开游玩。

## 菜摊敲敲乐亮点

- 反应点击玩法，敲掉坏菜和小偷，避开好菜。
- 连击、失误、专注时间和本地最佳记录构成完整反馈。
- 目标弹出节奏优化，开局可立即看到可点击目标。
- 支持鼠标和触屏，适合移动端短局游玩。

## 灯阵记忆亮点

- 灯序记忆玩法，每轮新增一盏灯，按顺序点回去。
- 4 色灯阵、音效反馈、重放扣分和本地最佳分数。
- 支持鼠标、触屏和键盘 1/2/3/4。
- DOM 按钮实现，无需 Canvas，也保持单文件自包含。

## 弹弓风暴亮点

- 参考经典弹弓抛射与结构破坏节奏，改成原创晶核塔题材。
- 5 个关卡，有限弹丸、目标清场、星级评分和关卡跳转。
- Matter.js 刚体物理驱动，木、玻璃、石块、爆桶和晶核会真实碰撞倒塌。
- 冲击星、裂光星、磁涡星、爆燃星四类弹丸，各有飞行中主动技能。
- 每关有不同风场，可用有限次数切换风向，增加解谜路线选择。
- 支持鼠标、触屏、键盘技能、暂停、重开和下一关流程。

## 晶塔前线亮点

- 轻量原创 MOBA 原型，包含三路线、兵线刷新、防御塔和双方晶核。
- 玩家英雄有普攻、Q/W/E/R 四个技能、能量、等级、经验和复活流程。
- 小兵会沿路线推进并自动索敌，防御塔和晶核会自动攻击敌方单位。
- 敌方英雄和友方 AI 会推进、索敌、释放技能和低血量回撤。
- 击杀获得金币和经验，可购买攻击、生命和冷却三类补给强化。
- 胜负目标明确：摧毁敌方晶核获胜，己方晶核被摧毁失败。

## 漂移赛道亮点

- 俯视角 2D 漂移竞速，3 圈比赛，包含 AI 对手和排名。
- 漂移会积累分数、倍率和氮气，连续控制越稳收益越高。
- 弯心剪裁点、加速带和胎温系统让每圈都有路线选择。
- HUD 显示速度、名次、圈速、氮气、漂移分、倍率和胎温。
- 支持键盘和触屏按钮，内置小地图、暂停、重开和结算。

## 松果快跑亮点

- 林间三线跑酷，收集松果并躲避木栏、枝门和来车。
- 跳跃、下滑、左右换线形成连续反应节奏。
- 磁铁、护盾、冲刺三类临时道具。
- 今日任务、本地最佳距离、最高分和累计收集记录。
- 支持键盘和触屏方向按钮。

## 雪岭快逃亮点

- 雪坡滑行竞速，沿地形起伏加速、跳跃和落地。
- 旋转、干净落地、冲刺和坡度共同影响速度与得分。
- 石块、断坡、松果收集和临时滑具形成路线变化。
- 今日任务、本地最佳距离、最高分和累计收集记录。
- 支持键盘和触屏旋转、跳跃、冲刺按钮。

## 霓虹贪吃蛇亮点

- 三个难度：轻松、街机、高手。
- 关卡推进：每吃 5 个食物进入下一关。
- 街机和高手模式会生成岩块障碍。
- 金果、蓝晶、护盾三类特殊道具。
- 连击、狂热、慢动作、粒子反馈和本地最高分。
- 支持键盘、WASD、触屏滑动和屏幕方向键。

## 星团大作战亮点

- 开始界面可选择自由模式、团队战、生存模式和大逃杀。
- 自由模式：100 人同场，12 分钟限时，死亡后复活，按个人质量结算。
- 团队战：10 队 5 人，12 分钟限时，队友可接分身，按队伍总质量结算。
- 生存模式：9 分钟生命制，吞噬可增加生命，按击杀、剩余生命和质量排名。
- 大逃杀：100 人开局，一命制，安全区持续收缩，活到最后获胜。
- 7600 像素大地图，后期食点刷新更快，战场会越来越密。
- 最大 16 个分身，支持分裂突袭、长按 W 连续吐球和冷却后自动合体。
- 有小刺和大刺两类刺球，触碰后会消失并重生，炸刺会先增加质量再分裂。
- AI 会追击、逃跑、进圈、分裂攻击、吐球、借刺反打、围猎、护航、接队友分身并给主力喂球。
- 后期会按可见区域绘制食点和细胞，并自动降低部分阴影、标签、小地图刷新和粒子细节来稳定帧率。
- 全屏竞技场、排行榜、小地图、粒子反馈、移动端触控和移动摇杆。

## 霓虹装甲亮点

- 五个模式：装甲突围、守点压制、精英猎杀、计时死斗、终局竞技。
- WASD 移动，鼠标瞄准，炮塔和车身独立转向。
- 可破坏砖墙、不可破坏钢墙、爆炸油桶和反弹炮弹。
- 轻坦、重坦、火箭车、布雷车和最终移动堡垒。
- 敌人有正面装甲、侧甲和后甲判定，绕后命中更痛。
- 每波结束三选一升级：穿甲弹、霰裂炮口、无人机、护盾、磁暴地雷等。
- 计时死斗为 16 辆坦克持续复活抢分，终局竞技为 16 辆坦克一命决斗。
- 竞技模式不弹出升级选择，地图模块箱拾取后会自动装配到坦克上。
- 地图改为结构化竞技场：中心开放、钢墙反弹点、砖墙断带和少量油桶。
- 竞技 AI 会互相攻击、躲避来弹、绕路找射线、抢模块箱并自动装配。
- 本地最高分、粒子爆炸、移动端触控、移动摇杆和暂停/重开。

## 霓虹 2048 亮点

- 经典 4x4 2048 合成规则，方向键、WASD 和触屏滑动都可操作。
- 首次合成出 4、8、16 等新数字时奖励一个随机工具。
- 每 100 步额外奖励一个随机工具，节奏比早期版本更克制。
- 撤回、锤子、交换、冻结、炸弹、升阶、共鸣、裂变、磁吸、回收十种工具。
- 有效滑动会播放方块滑行动画，合成和新生成方块有弹出反馈。
- 本地最高分、移动端布局和结束统计。

## 山海伏魔录亮点

- 卡牌肉鸽原型，包含地图节点、战斗、事件、歇脚和 Boss。
- 有攻击、防御、过牌、易伤、虚弱、反震等卡牌效果。
- 战斗胜利可选牌，精英和事件会获得遗物。
- 8 层流程，牌组、金币、血量、遗物和升级共同构筑。

## 薯仔突围亮点

- 自动射击生存玩法，移动走位，武器自动索敌。
- 5 分钟流程，精英刷新，最终首领登场。
- 多武器和升级选项，包含子弹、近战、电击、范围效果等。
- 敌人掉落经验、晶体和补给，支持键盘、鼠标/触控移动，移动端可用左下摇杆。

## 怪潮逆塔防亮点

- 反向塔防玩法，玩家负责组织怪潮进攻英雄防线。
- 三条进攻路线、怪物契约、怪潮队列和战场邪术共同构成编队策略。
- 英雄 AI 会根据上一波怪潮补塔、升级和调整目标，后续夜晚逐步反制。
- 每夜结束可选择异变强化，形成持续进化的怪潮构筑。

## 萝卜边境亮点

- 传统塔防玩法，放置弩塔、泡泡塔、萝卜炮等守住萝卜生命。
- 多地图、多难度、障碍清理、炮塔升级/出售和技能支援。
- 没有敌人时炮塔会自动清理障碍，形成经济和建造节奏。
- 8 波流程、评分结算和规则弹窗，适合轻量单局游玩。

## 霓虹贪吃蛇操作

- 方向键 / WASD：移动。
- 空格：暂停或继续。
- 点击右上角按钮：暂停、静音、重新开始。

## 星团大作战操作

- 鼠标或手指：控制方向，移动端可用左下摇杆。
- 空格：分裂。
- W：按住连续吐球。
- P：暂停或继续。
- R：重新开始。

## 霓虹装甲操作

- WASD / 方向键：移动。
- 移动端左下摇杆：移动。
- 鼠标或手指：瞄准。
- 鼠标左键 / 空格：开火。
- 鼠标右键 / Shift / E：释放电磁脉冲。
- B：在竞技模式中提示模块自动装配规则。
- P：暂停或继续。
- R：重新开始。

## 霓虹 2048 操作

- 方向键 / WASD / 滑动：移动方块。
- 点击工具按钮：选择或使用工具。
- 锤子 / 炸弹 / 交换 / 升阶 / 共鸣 / 裂变：选择工具后点击棋盘格。
- 磁吸 / 回收 / 冻结 / 撤回：点击工具后立即生效。
- Esc：取消当前工具。
- Z：撤回一步。

## 山海伏魔录操作

- 点击地图节点推进流程。
- 点击手牌出牌，点击结束回合。
- 战斗后点击奖励、事件或歇脚选项继续。

## 薯仔突围操作

- WASD / 方向键：移动。
- 鼠标或触控：控制移动方向，移动端可用左下摇杆。
- 点击按钮：开始、暂停、新局和选择升级。

## 怪潮逆塔防操作

- 选择出怪路线，再点击怪物契约加入队列。
- 智能配队可自动填充队列，释放怪潮后观察英雄防线反应。
- 战场邪术可在选择路线后释放，夜晚结束后选择异变强化。
- 点击规则查看玩法说明，新局可重置流程。

## 萝卜边境操作

- 点击空地建塔，点击已有炮塔升级或出售。
- 开波前可切换地图和难度，开波后守住萝卜生命。
- 技能按钮可选择位置释放，速度和暂停按钮控制战斗节奏。
- 新局和规则按钮位于顶部。

## 弹弓风暴操作

- 鼠标或触屏拖拽弹丸蓄力，松开发射。
- 飞行中点击技能按钮或按空格触发当前弹丸技能。
- 点击换风按钮或按 Q 切换风向，P 暂停，R 重开。

## 晶塔前线操作

- WASD / 方向键移动，点击战场设置移动点或锁定敌方目标。
- Q 弧光刃、W 相位步、E 星沼、R 晶陨，侧栏技能按钮也可释放。
- 点击补给按钮购买强化，空格或 P 暂停，顶部按钮可重开。

## 漂移赛道操作

- W / 方向上：加速。
- A / D 或方向左 / 方向右：转向。
- S / 方向下：制动。
- Shift / 空格：氮气。
- P：暂停或继续，顶部按钮可暂停和重开。

## 松果快跑操作

- A / D 或方向左 / 方向右：换线。
- W / 方向上 / 空格：跳跃。
- S / 方向下：下滑。
- P：暂停或继续，触屏按钮可换线、跳跃和下滑。

## 雪岭快逃操作

- A / D 或方向左 / 方向右：空中旋转。
- W / 方向上 / 空格：跳跃或旋转。
- S / 方向下：压低重心。
- Shift：冲刺。
- P：暂停或继续，触屏按钮可旋转、跳跃和冲刺。

## 节拍饭团操作

- A / S / D / F：选择米、苔、鱼、汤。
- 点击底部食材按钮也可操作，节拍命中越准连击越高。
- P 暂停，R 重开，顶部按钮也可控制流程。

## 影邮潜行操作

- WASD / 方向键或屏幕方向按钮：移动邮差。
- 等待：原地过一回合；烟雾弹：本回合无视灯光。
- 撤回可退回上一步，重开可重置当前关卡。

## 夜市面摊操作

- 点击订单、锅位、浇头和出餐按钮完成订单。
- 数字键可快速选择锅位，顶部按钮可开摊、暂停或重开。
- 移动端直接点按各控件即可游玩。

## 棱镜果园操作

- 选择镜片工具后点击棋盘格放置或替换镜片。
- 拆除工具会返还镜片库存，点亮按钮用于检查光路。
- R 重开当前关卡。

## 字灵守门操作

- 直接键入画面中出现的字词来击退字灵。
- 点击击退按钮可释放充能击退，R 重开。
- 优先输入最靠近城门的目标。

## 道岔快线操作

- 点击画面道岔或按 1/2/3 切换方向。
- 红、蓝、绿列车要进入对应颜色站台。
- P 暂停，R 重开。

## 潮汐拾荒操作

- WASD / 方向键或屏幕方向按钮：移动潜水员。
- Shift 或冲刺按钮：短时冲刺。
- 回到水面会自动卖出货物并补满氧气。

## 像素谜格操作

- 选择填黑、画叉或擦除工具后点击格子。
- 鼠标右键可快速画叉，检查按钮可手动判定。
- 完成后可以进入下一题。

## 彗尾织者操作

- WASD / 方向键移动，Shift 或闪现按钮闪现。
- 移动端可拖动画布控制位置。
- 贴近弹幕擦弹得分，击破 Boss 获胜。

## 月面轻降操作

- ↑ / W：主推；←/→ 或 A/D：转向；↓ / S：自动扶正。
- 移动端可用底部按钮控制。
- 低速、低角度接触平台即可成功着陆。

## 菜摊敲敲乐操作

- 点击坏菜和小偷得分，别点带笑脸的好菜。
- 连击达到要求后可点专注按钮放慢目标速度。
- 失误满 5 次或时间结束后结算。

## 灯阵记忆操作

- 观察灯阵闪烁后，按同样顺序点回去。
- 键盘 1/2/3/4 对应四盏灯。
- 重放会扣少量分数，R 可重开。
