# 剪映与必剪二次编辑说明

## 可直接发布的文件

- `deliverables/mini-browser-games-promo-v2-hq-captioned.mp4`：93 秒横版 V2 推荐终版
- `deliverables/mini-browser-games-promo-v2-hq-clean.mp4`：横版 V2 无烧录字幕版
- `deliverables/mini-browser-games-promo-1080p-captioned.mp4`：96.5 秒横版终版
- `deliverables/mini-browser-games-promo-1080p-clean.mp4`：横版无烧录字幕版
- `deliverables/mini-browser-games-promo-vertical-captioned.mp4`：46 秒竖版终版
- `deliverables/mini-browser-games-promo-vertical-clean.mp4`：竖版无烧录字幕版

## 分层素材

- `deliverables/editable-v2/`：V2 深沉配音、混音与逐镜头字幕
- `deliverables/editable/clips/`：逐游戏 H.264 实机片段
- `deliverables/editable/audio/voice-master.wav`：横版纯配音
- `deliverables/editable/audio/neon-drive-bgm.wav`：原创 BGM
- `deliverables/editable/audio/transitions-sfx.wav`：转场与冲击音效
- `deliverables/editable/audio/master-mix.wav`：横版完整混音
- `deliverables/editable/audio/voice-vertical.wav`：竖版纯配音
- `deliverables/editable/subtitles/`：SRT 与 ASS 字幕
- `deliverables/editable/voice-samples/`：三种备用配音样音

## 在剪映或必剪中修改

1. 新建 1920×1080 横版或 1080×1920 竖版工程。
2. 若只需删减内容，直接导入 `clean.mp4`，再导入 SRT 字幕即可。
3. 若要替换某款游戏，导入 `clips/` 中的独立片段覆盖对应时间段。
4. 若要修改音量，分别导入配音、BGM 和音效 WAV，不使用已经合成的 `master-mix.wav`。
5. 建议配音峰值保持在 -3 dB 左右，BGM 在人声期间降低约 5 dB。

横版核心时间码：`00:05` 星团、`00:20` 星炉、`00:40` 其他代表作、`01:11` 类型蒙太奇、`01:28` 结尾。
